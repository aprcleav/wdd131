WDD131 Project Site Plan

Site Title:

April Cleaver Illustration

It seems to be common practice for an artist to title their portfolio website with only their name, or their name plus keywords like "illustration" or "art & design". Since I will be using my own artwork for the website content, I chose my own name plus the "illustration" keyword.

Purpose:

This website acts as an artist's portfolio. People can come here to learn more about the artists training, experience, and style. They can read updates about recent projects the artist has worked on, and contact the artist to inquire about collaboration. Pages will include home, about, blog, contact, and illustrations.

Scenarios:

  1. Does this artist's style fit our needs?
  2. Does the artist's work show consistency?
  3. Can information about the artist be easily navigated?

Color Schema

  • Header & footer background: Royal Gold (#FFE066)
  • Call to action background: Periwinkle (#BCB6FF)
  • Accent color for hover effects: Icy Aqua (#C0FDFB)
  • Text color, button background color: Coffee Bean (#161019)
  • Text color for buttons: White (#fff)

Typography

  • Headings: Poppins ExtraBold 800
  • Body: Poppins Regular 400
  • Quotes: Noto Serif Regular 600 Italic
